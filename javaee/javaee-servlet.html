# Servlet С╗╗тіА

Servlet ТЅДУАїС╗ЦСИІСИ╗УдЂС╗╗тіА№╝џ

  • У»╗тЈќт«бТѕиуФ»№╝ѕТхЈУДѕтЎе№╝ЅтЈЉжђЂуџёТўЙт╝ЈуџёТЋ░ТЇ«сђѓУ┐ЎтїЁТІгуйЉжАхСИіуџё HTML УАетЇЋ№╝їТѕќУђЁС╣ЪтЈ»С╗ЦТў»ТЮЦУЄф applet ТѕќУЄфт«џС╣Ѕуџё HTTP т«бТѕиуФ»уеІт║ЈуџёУАетЇЋсђѓ
  • У»╗тЈќт«бТѕиуФ»№╝ѕТхЈУДѕтЎе№╝ЅтЈЉжђЂуџёжџљт╝Јуџё HTTP У»иТ▒ѓТЋ░ТЇ«сђѓУ┐ЎтїЁТІг cookiesсђЂтфњСйЊу▒╗тъІтњїТхЈУДѕтЎеУЃйуљєУДБуџётјІу╝ЕТа╝т╝ЈуГЅуГЅсђѓ
  • тцёуљєТЋ░ТЇ«т╣ХућЪТѕљу╗ЊТъюсђѓУ┐ЎСИфУ┐ЄуеІтЈ»УЃйжюђУдЂУ«┐жЌ«ТЋ░ТЇ«т║Њ№╝їТЅДУАї RMI Тѕќ CORBA У░Ѓуће№╝їУ░Ѓуће Web ТюЇтіА№╝їТѕќУђЁуЏ┤ТјЦУ«Ау«ЌтЙЌтЄ║т»╣т║ћуџётЊЇт║ћсђѓ
  • тЈЉжђЂТўЙт╝ЈуџёТЋ░ТЇ«№╝ѕтЇ│ТќЄТАБ№╝Ѕтѕ░т«бТѕиуФ»№╝ѕТхЈУДѕтЎе№╝ЅсђѓУ»ЦТќЄТАБуџёТа╝т╝ЈтЈ»С╗ЦТў»тцџуДЇтцџТаиуџё№╝їтїЁТІгТќЄТюгТќЄС╗Х№╝ѕHTML Тѕќ XML№╝ЅсђЂС║їУ┐ЏтѕХТќЄС╗Х№╝ѕGIF тЏЙтЃЈ№╝ЅсђЂExcel уГЅсђѓ
  • тЈЉжђЂжџљт╝Јуџё HTTP тЊЇт║ћтѕ░т«бТѕиуФ»№╝ѕТхЈУДѕтЎе№╝ЅсђѓУ┐ЎтїЁТІгтЉіУ»ЅТхЈУДѕтЎеТѕќтЁХС╗ќт«бТѕиуФ»УбФУ┐ћтЏъуџёТќЄТАБу▒╗тъІ№╝ѕСЙІтдѓ HTML№╝Ѕ№╝їУ«Йуй« cookies тњїу╝ЊтГўтЈѓТЋ░№╝їС╗ЦтЈітЁХС╗ќу▒╗С╝╝уџёС╗╗тіАсђѓ

# init() Тќ╣Т│Ћ

init Тќ╣Т│ЋУбФУ«ЙУ«АТѕљтЈфУ░ЃућеСИђТгАсђѓт«ЃтюеуггСИђТгАтѕЏт╗║ Servlet ТЌХУбФУ░Ѓуће№╝їтюетљју╗ГТ»ЈТгАућеТѕиУ»иТ▒ѓТЌХСИЇтєЇУ░ЃућесђѓтЏаТГц№╝їт«ЃТў»ућеС║јСИђТгАТђДтѕЮтДІтїќ№╝їт░▒тЃЈ Applet уџё init Тќ╣Т│ЋСИђТаисђѓ

Servlet тѕЏт╗║С║јућеТѕиуггСИђТгАУ░Ѓућет»╣т║ћС║јУ»Ц Servlet уџё URL ТЌХ№╝їСйєТў»ТѓеС╣ЪтЈ»С╗ЦТїЄт«џ Servlet тюеТюЇтіАтЎеуггСИђТгАтљ»тіеТЌХУбФтіаУййсђѓ

тйЊућеТѕиУ░ЃућеСИђСИф Servlet ТЌХ№╝їт░▒С╝џтѕЏт╗║СИђСИф Servlet т«ъСЙІ№╝їТ»ЈСИђСИфућеТѕиУ»иТ▒ѓжЃйС╝џС║ДућЪСИђСИфТќ░уџёу║┐уеІ№╝їжђѓтйЊуџёТЌХтђЎуД╗С║цу╗Ў doGet Тѕќ doPost Тќ╣Т│Ћсђѓinit() Тќ╣Т│Ћу«ђтЇЋтю░тѕЏт╗║ТѕќтіаУййСИђС║ЏТЋ░ТЇ«№╝їУ┐ЎС║ЏТЋ░ТЇ«т░єУбФућеС║ј Servlet уџёТЋ┤СИфућЪтЉйтЉеТюЪсђѓ

  // тѕЮтДІтїќС╗БуаЂ...

# service() Тќ╣Т│Ћ

service() Тќ╣Т│ЋТў»ТЅДУАїт«ъжЎЁС╗╗тіАуџёСИ╗УдЂТќ╣Т│ЋсђѓServlet т«╣тЎе№╝ѕтЇ│ Web ТюЇтіАтЎе№╝ЅУ░Ѓуће service() Тќ╣Т│ЋТЮЦтцёуљєТЮЦУЄфт«бТѕиуФ»№╝ѕТхЈУДѕтЎе№╝ЅуџёУ»иТ▒ѓ№╝їт╣ХТііТа╝т╝ЈтїќуџётЊЇт║ћтєЎтЏъу╗Ўт«бТѕиуФ»сђѓ

Т»ЈТгАТюЇтіАтЎеТјЦТћХтѕ░СИђСИф Servlet У»иТ▒ѓТЌХ№╝їТюЇтіАтЎеС╝џС║ДућЪСИђСИфТќ░уџёу║┐уеІт╣ХУ░ЃућеТюЇтіАсђѓservice() Тќ╣Т│ЋТБђТЪЦ HTTP У»иТ▒ѓу▒╗тъІ№╝ѕGETсђЂPOSTсђЂPUTсђЂDELETE уГЅ№╝Ѕ№╝їт╣ХтюежђѓтйЊуџёТЌХтђЎУ░Ѓуће doGetсђЂdoPostсђЂdoPut№╝їdoDelete уГЅТќ╣Т│Ћсђѓ

СИІжЮбТў»У»ЦТќ╣Т│ЋуџёуЅ╣тЙЂ№╝џ

public void service(ServletRequest request,
                    ServletResponse response)
      throws ServletException, IOException{
